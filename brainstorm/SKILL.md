# Brainstorm

## Befehle

- **/IDEAS20 [topic]** — Zwanzig verschiedene Ideen
- **/GIFTS [person + occasion + budget]** — Geschenkoptionen
- **/NAMES [thing + vibe]** — Name-Ideen im gewünschten Stil
- **/UNUSUAL [problem]** — Zehn unkonventionelle Ansätze
- **/ANGLE [topic]** — Übersehener innovativer Winkel
- **/COMBINE [A + B]** — Verbindet zwei Konzepte in fünf Richtungen
- **/METAPHOR [concept]** — Kreative Metapher für Konzept
- **/STARTERS [topic]** — Nicht-langweilige Gesprächseröffnungen
- **/JOURNALSO [theme]** — Zehn Tagebuch-Prompts zum Schreiben
- **/AS [role + problem]** — Löst Problem aus Perspektive einer Profession
- **/CHILD [problem]** — Wie würde ein neugieriges Kind es lösen

## Use-Cases

- Neuer Produktname: `/NAMES "Fitness-App" "motivierend"` generiert Optionen
- Stuck bei Problemlösung: `/UNUSUAL` + `/AS "Engineer"` bringt frische Perspektive
- Social-Media-Kampagne: `/IDEAS20` + `/ANGLE` für originelle Ansätze
- Geschenk suchen: `/GIFTS "Chef, Geburtstag, 50€"` mit klugen Vorschlägen
