# Small Business

## Befehle

- **/INVOICE [client + amount + date]** — Professionelle Rechnung strukturieren
- **/CUSTOMER-EMAIL [situation + goal]** — Kundenmail schreiben
- **/CASHFLOW [monthly revenue/expenses]** — Cashflow-Planung erstellen
- **/CONTRACT [agreement type + terms]** — Vertragsentwurf vorbereiten
- **/PROPOSAL [service + client + budget]** — Verkaufsangebot schreiben
- **/REMINDER [overdue amount + client]** — Zahlungserinnerung professionell
- **/FEEDBACK [customer + service]** — Kundenfeedback-Anfrage
- **/ONBOARDING [product/service]** — Kundeneinführungs-Checkliste
- **/UPSELL [current customer + product]** — Upsell-Vorschlag-Sequenz
- **/RETENTION [churn risk]** — Kundenhalte-Strategie
- **/FOLLOWUP [meeting/proposal]** — Follow-up-Email-Sequenz
- **/EXPENSE [cost category + amount]** — Ausgaben-Kategorisierung

## Use-Cases

- Rechnung schreiben: `/INVOICE "Acme Corp, €2500, Juni 2026"` + `/REMINDER "30 Tage überfällig, €2500"` für Zahlungsfolge
- Neuer Kunde gewinnen: `/PROPOSAL + `/FOLLOWUP` orchestrieren Verkaufsprozess
- Monatliche Planung: `/CASHFLOW "€15k Revenue, €8k Ausgaben"` für Liquiditäts-Überblick
- Kundenbeziehung sichern: `/ONBOARDING` + `/FEEDBACK` + `/RETENTION` für Kundenerlebnis
- Geschäft skalieren: `/UPSELL` + `/CONTRACT` für zusätzliche Einnahmen
